
# Hemant Kumar — Solana & Rust Developer

I didn't start in blockchain. I started building full-stack web apps —
React, Node.js, the usual. Then I built a token launchpad on Solana,
and everything changed.

What got me wasn't just the tech. It was the mindset shift: write a
great contract, deploy it, and it runs forever. No servers. No AWS
bills. No DevOps at 3am. Users own their data — not a company sitting
on a private database. That's how software should work.

Now I go deep on Solana and Rust. I care about compute unit efficiency,
PDA security, account validation, and building DeFi primitives that are
hard to break. Rust forces you to be honest — no hidden costs, no
garbage collector surprises, just deterministic and fast.

I'm looking for on-chain engineering roles — specifically Solana smart
contract development, DeFi protocol work, or anything pushing the edges
of what's possible on-chain.
